Zacit

Tlacitka

Bootstrap umoznuje snadne stylovani tlacitek pomoci trid jako napr. btn-primary

                
                    <button class="btn btn-primary">Primary button</button>
                    <button class="btn btn-warning">Warning button</button>

Karty

Karta v Bootstrapu je super zpusob jak struktorovat obsah.

            
                <div class="card mx-auto">   
                    <div class="card-body">
                        <h3 class="card-title">Ukazkova karta</h3>
                        <p class = "card-text">Toto je ukazka karty v Bootstrapu.</p>
                        <a href="#" class="btn btn-info">Detail</a>
                    </div>
                </div>
